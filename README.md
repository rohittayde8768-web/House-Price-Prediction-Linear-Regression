# 🏠 House Price Prediction using Linear Regression

## 📌 Project Overview
This project aims to predict house prices based on property features using Linear Regression.

The project covers:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Selection
- Model Training
- Model Evaluation
- Visualization

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## 📊 Model Evaluation Metrics
- R² Score
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)

---

## ▶️ How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the model:
   python src/train_model.py
